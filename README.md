# simplejdbc
Ejemplo simple usando jdbc con postgresql

### Dependencias
* Java 21
* docker / podman
* docker-compose / podman-compose (necesita configuracion)

### Extensiones visual code recomendadas
* Java Language Support
* Extension Pack for Java Auto Config
* Spring Boot Tools
* Spring Initializr Java Support
* Lombok Annotations
* Container Tools

### Listado de endpoints
http://localhost:8080/api/v1/swagger-ui/index.html
